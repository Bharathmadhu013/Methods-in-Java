# Methods in Java

This repository contains examples of different types of methods in Java. The programs are implemented using simple logic and dynamic inputs to help beginners understand method concepts effectively.

## 📌 What is a Method?

A method is a block of code that performs a specific task. Methods improve code reusability, readability, and maintainability.

### Syntax

```java
returnType methodName(parameters) {
    // statements
    return value;
}
```

---

## Types of Methods in Java

### 1. Static Method Without Parameters and Without Return Type

```java
static void m1() {
}
```

### 2. Static Method With Parameters and Without Return Type

```java
static void m2(int a) {
}
```

### 3. Static Method Without Parameters and With Return Type

```java
static String m3() {
    return "";
}
```

### 4. Static Method With Parameters and With Return Type

```java
static int m4(String s) {
    return 0;
}
```

### 5. Non-Static Method Without Parameters and Without Return Type

```java
void m5() {
}
```

### 6. Non-Static Method With Parameters and Without Return Type

```java
void m6(char c) {
}
```

### 7. Non-Static Method Without Parameters and With Return Type

```java
float m7() {
    return 0.0f;
}
```

### 8. Non-Static Method With Parameters and With Return Type

```java
float m8(char c) {
    return 0.0f;
}
```
